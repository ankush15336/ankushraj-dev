    If a > b, then a + c > b + c

    If a > b and c > 0, then a √ó c > b √ó c

    1. Basic Concepts
    Cost Price (CP): Price at which article is purchased
    Selling Price (SP): Price at which article is sold
    Marked Price (MP): Price printed on article

    Profit: When SP > CP ‚Üí Profit = SP - CP
    Loss: When SP < CP ‚Üí Loss = CP - SP
    Example: Pen bought for ‚Çπ80, sold for ‚Çπ100
    Profit = 100 - 80 = ‚Çπ20
    2. Profit and Loss Percentage
    Profit % = (Profit/CP) √ó 100
    Loss % = (Loss/CP) √ó 100

    SP = CP √ó (100+P%)/100 [Profit]
    SP = CP √ó (100-L%)/100 [Loss]

    CP = (SP √ó 100)/(100+P%)
    CP = (SP √ó 100)/(100-L%)
    Ex 1: CP=‚Çπ500, SP=‚Çπ600. Profit %?
    Profit = 100, Profit % = (100/500)√ó100 = 20% ‚úì

    Ex 2: CP=‚Çπ800, Loss=10%. SP?

    Ex 3: SP=‚Çπ900, Profit=20%. CP?
    CP = (900√ó100)/120 = ‚Çπ750 ‚úì
    3. Dishonest Shopkeeper

    (x = false weight in grams)

    Profit % = [(1000-900)/900]√ó100 = 11.11% ‚úì
    4. Discount
    Discount % = (Discount/MP) √ó 100
    SP = MP √ó (100-D%)/100
    Ex: MP=‚Çπ1000, Discount=20%
    SP = 1000 √ó 0.8 = ‚Çπ800 ‚úì
    5. Successive Discounts
    Single = d‚ÇÅ + d‚ÇÇ - (d‚ÇÅ√ód‚ÇÇ)/100
    Ex: 20% and 10%
    Single = 20+10-(20√ó10)/100 = 28% ‚úì
    üí° Quick: 0.8 √ó 0.9 = 0.72 ‚Üí Discount = 28%
    6. Marked Price with Profit
    Ex: CP=‚Çπ500, MP 40% above, sold at 20% discount

    üìä Chapter 1.8 ‚Äî Average & Weighted Average
    1. Concept of Average
    Average: Sum of all values divided by number of values

    Average = Sum of all values / Total number of values
    Ex: Find average of 10, 20, 30, 40, 50
    Sum = 150, Count = 5
    Average = 150/5 = 30 ‚úì
    2. Average with Addition/Removal
    When one item added:
    New Sum = Old Sum + New Item

    When one item removed:
    New Sum = Old Sum - Removed Item
    Ex: Average of 5 numbers is 20. One number 25 added. New average?

    Old Sum = 5 √ó 20 = 100
    New Sum = 100 + 25 = 125
    New Average = 125/6 = 20.83 ‚úì
    3. Weighted Average
    Weighted Average: Average where some values have more importance (weight) than others
    Weighted Average = (w‚ÇÅ√óv‚ÇÅ + w‚ÇÇ√óv‚ÇÇ + w‚ÇÉ√óv‚ÇÉ...) / (w‚ÇÅ + w‚ÇÇ + w‚ÇÉ...)

    Where: w = weight, v = value
    Ex: Class A (30 students, avg 70), Class B (20 students, avg 80). Overall average?

    Weighted Avg = (30√ó70 + 20√ó80)/(30+20)
    = (2100 + 1600)/50
    = 3700/50 = 74 ‚úì
    4. Average Change Problems
    Ex: Average of 10 numbers is 50. If each increased by 5, new average?

    Each increased by 5 ‚Üí Average also increases by 5
    New Average = 50 + 5 = 55 ‚úì
    üí° RULE:
    If all values increase/decrease by x ‚Üí Average increases/decreases by x
